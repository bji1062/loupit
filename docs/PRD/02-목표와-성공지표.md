# 목표와 성공지표

**목적**: loupit의 비즈니스/제품 목표를 정의하고, 각 목표를 검증할 수 있는 성공지표(KPI)와 정량 목표치를 확정한다.

**참조 브리프 섹션**: §1(제품 정체성), §2(핵심 결정 5·6·7·8), §3(MVP 범위), §8(기술 스택).
**상위 추적**: 본 문서는 PRD 항목이며 하위 문서(USECASE/FRD)가 여기의 목표 ID(G*)와 지표 ID(GM*)를 인용한다.

---

## 1. 목표 체계 개요

- **북극성 지표(North Star)**: 검색 유입 기반 **월 순방문(UV)**. 광고 노출·수익·재방문이 모두 여기서 파생되므로 단일 성장 축으로 삼는다(§2-7 "SEO가 수익의 뼈대").
- **인과 사슬**: SEO 색인 확대(G2) → organic UV 증가(GM1) → 광고 노출/수익(G1·G3) 증가. 성능(G5)과 비교 완료율(G4)이 색인·체류·재방문을 뒷받침한다.
- 모든 정량 목표치에는 **(가정)** 표기를 붙인다. 브리프에 확정 수치가 없으므로 신규 사이트 기준 합리적 추정이며, 출시 후 실측으로 보정한다.

---

## 2. 비즈니스 목표

| ID | 목표 | 근거 결정 | 성공 판정 |
|----|------|-----------|-----------|
| **G1** | 광고·제휴로 사이트를 수익화한다 (유일 수익원) | §1, §2-5 AdSense 자동+수동 슬롯 + affiliate | AdSense 승인 후 월 광고+제휴 매출이 발생하고 UV 성장에 비례해 증가 |

**G1 수익 모델(가정, 방향성 참고용)**: 월매출 ≈ 월 PV × 광고밀도 × RPM.

| 시점 | 월 organic UV(가정) | 월 PV(가정, UV×1.8) | 페이지RPM(가정) | 월 광고매출 개략(가정) |
|------|------|------|------|------|
| 출시+3개월(M3) | 5,000 | 9,000 | \$1.5 | ≈ \$14 |
| 출시+6개월(M6) | 20,000 | 36,000 | \$1.5 | ≈ \$54 |
| 출시+12개월(M12) | 60,000 | 108,000 | \$2.0 | ≈ \$216 |

> 위 매출 수치는 규모 감을 위한 예시이며 **커밋 목표가 아니다**. 커밋 대상은 §4의 KPI 목표치다.

| **G3** | AdSense 정책 심사를 통과하고 광고를 유효 노출시킨다 | §2-5·6 | AdSense 승인 획득 + 광고 뷰어빌리티 확보(GM3·GM4) |

| **G6** | 재방문을 유도한다 ("최근 비교" localStorage 활용) | §2-2 localStorage 저장 | 재방문 사용자 비율 확보(GM8) |

## 4. 성공지표(KPI)

측정 도구 표기: **GA4**(Google Analytics 4, 클라이언트 이벤트), **GSC**(Google Search Console), **AdSense**(AdSense 리포트), **PSI/CrUX**(PageSpeed Insights·Chrome UX Report). **측정 도구 선정 자체는 브리프 미확정 사항이므로 (가정)이다**(GA4는 서버 저장 없는 §2-2 정책과 호환되는 클라이언트 분석으로 가정).

| ID | KPI | 정의 | 측정 도구 | 목표치(모두 가정) | 관련 목표 |
|----|-----|------|-----------|-------------------|-----------|
| **GM1** | 월 순방문(UV) | organic 채널 월간 순 사용자 수 | GA4 | M3 ≥ 5,000 / M6 ≥ 20,000 / M12 ≥ 60,000 | G1·G2 (북극성) |
| **GM2** | 검색 색인 페이지 수 | GSC "색인 생성됨" 페이지 수 | GSC | M3 ≥ 150 / M6 ≥ 190 (총 콘텐츠 156~206 페이지 대비 M6 색인율 ≥ 95%) | G2 |
| **GM3** | AdSense 승인 | 게시자 계정 콘텐츠 심사 통과 여부(이진) | AdSense | M2까지 승인 완료 | G3 |
| **GM4** | 광고 노출·뷰어빌리티 | 광고 노출수 및 Active View 뷰어빌리티(%) | AdSense | 뷰어빌리티 ≥ 70% (노출수는 PV에 종속) | G1·G3 |
| **GM5** | 비교 완료율 | (비교 리포트 생성 이벤트 세션 / 비교 입력 시작 이벤트 세션) | GA4 이벤트 | ≥ 40% | G4 |
| **GM6** | 이탈률 | 콘텐츠 페이지 이탈률(단일 페이지·무참여 세션 비율) | GA4 | 콘텐츠 페이지 ≤ 70% (참여율 ≥ 30%) | G2·G4 |
| **GM7** | Core Web Vitals | 필드데이터 75백분위 값 (임계치는 Google 표준) | PSI/CrUX·GSC | 3개 지표 모두 "양호" 75%+: LCP ≤ 2.5s, INP ≤ 200ms, CLS ≤ 0.1 | G5 |
| **GM8** | 재방문 | 30일 내 재방문 사용자 비율 | GA4 | ≥ 15% | G6 |

**보조 지표(참고, 목표치 미커밋)**: 페이지당 광고 CTR, 제휴 링크 클릭수, 회사 상세 → "이 회사로 비교하기" CTA 클릭률(§3 F4), 검색 API 사용률.

## 5. 가정과 검증 조건

| 항목 | 가정 내용 | 검증/보정 방법 |
|------|-----------|----------------|
| 콘텐츠 총량 | 회사 ~96(§3 F4, 복지 데이터 보유 회사) + 인기 조합 페이지 약 50(§3 F5, 수치 브리프 미확정) + 정책·랜딩 약 10 = 약 156~206 페이지 | 빌드 산출물·sitemap.xml 실측으로 확정 |
| UV 성장 곡선 | 신규 SEO 사이트가 색인 정착 후 3~12개월에 걸쳐 로그 성장 | 출시 후 월별 GA4/GSC 실측으로 커브 보정 |
| RPM \$1.5~2.0 | 한국어 콘텐츠·범용 주제 AdSense 페이지RPM 범위 추정 | 승인 후 AdSense 실측으로 대체 |
| 뷰어빌리티 70% | 콘텐츠 상단·인접 슬롯 배치(§2-6) 기준 업계 통상치 | AdSense Active View 실측 |
| GA4 채택 | 서버 저장 없는(§2-2) 클라이언트 분석 도구로 GA4 사용 | 최종 도구는 SPEC 단계에서 확정 |
| AdSense 승인 전제 | 오리지널 콘텐츠 150+ 페이지와 정책 페이지 완비(§3 F7)가 승인 요건을 충족 | 심사 결과로 검증, 반려 시 콘텐츠 보강 |

**리스크 메모**: 광고 밀도를 높이면 GM4(노출)는 오르나 GM6(이탈)·GM7(CLS)이 악화될 수 있다. §2-6 광고 배치 정책(비교 입력 화면 광고 최소화, 콘텐츠 페이지 위주)을 지표 간 상충의 조정 기준으로 삼는다.
