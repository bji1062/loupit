# USECASE — loupit

> 이 문서는 **얇은 인덱스**다. 요약과 참조 파일 목록, 유즈케이스 마스터표(UC-ID)·F1~F8 커버리지 매트릭스만 담는다. 개별 UC의 액터·사전조건·기본 흐름·대안 흐름 등 상세 서술은 각 `USECASE/xx.md` 파일에 있다.

## 1. 목적

loupit 비교 도구·정적 콘텐츠·읽기 전용 API에 대한 사용자·시스템 상호작용을 표준 유즈케이스로 정의한 문서군의 진입점이다. 각 UC는 PRD 기능 요구사항 F1~F8과 타깃 사용자(PRD-SEG-*/PRD-PER-*/PRD-CTX-*)를 인용하여 USECASE→PRD 추적성을 유지한다(정의·명명 규칙은 [USECASE/01-액터와-개요.md](USECASE/01-액터와-개요.md)). 로그인/회원/프로파일러/서버 측 사용자 데이터 쓰기는 제품 범위에서 영구 제외이므로 어떤 흐름에도 등장하지 않는다.

## 4. F1~F8 → UC 커버리지 매트릭스

각 기능이 어떤 UC로 커버되는지의 지도다. **주 커버**는 해당 기능을 직접 다루는 UC, **연동·엣지**는 다른 기능 대역이나 횡단 대역에서 해당 기능을 인용·보조·오류 처리하는 UC다. F8은 전용 십단위 대역 없이 소비 맥락별 UC로 흡수된다([01](USECASE/01-액터와-개요.md) §4.1).

| 기능(F) | 주 커버 UC | 연동·엣지 UC |
| --- | --- | --- |
| **F1** 회사 검색·선택 | UC-10, UC-11, UC-12, UC-13, UC-14, UC-15 | UC-A1(참조 로드), UC-43·UC-52(슬롯 프리필·연동), UC-91(검색 API 실패) |
| **F2** 비교 입력 | UC-20, UC-21, UC-22, UC-23, UC-24, UC-25 | UC-14·UC-15(F1→F2), UC-43·UC-52(프리필·진입), UC-30·UC-35(입력 연동), UC-92·UC-93(엣지) |
| **F3** 클라이언트 비교 리포트 | UC-30, UC-31, UC-32, UC-33, UC-34, UC-35, UC-36 | UC-A1(번들), UC-A2(로컬 저장), UC-52(재계산), UC-90·UC-92·UC-93(엣지) |
| **F4** 회사 상세 페이지 | UC-40, UC-41, UC-42, UC-43, UC-44 | UC-A3(색인), UC-A4(생성·배포), UC-53(내부 탐색), UC-93·UC-94·UC-95(엣지) |
