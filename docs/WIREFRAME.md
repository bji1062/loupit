# WIREFRAME — loupit

> 이 문서는 **얇은 인덱스**다. 요약과 참조 파일 목록, 화면(SCR) 마스터표, 광고 배치 요약만 담는다. 각 화면의 저충실도 ASCII 와이어프레임·구획 설명·상태 변형 등 상세는 각 `WIREFRAME/xx.md` 파일에 있다.

## 1. 목적

FLOW가 확정한 화면·플로우(SCR·구획)를 **모바일 우선 저충실도(lo-fi) 와이어프레임**으로 구체화한 문서군의 진입점이다. 각 화면은 WIREFRAME → FLOW → FRD → USECASE → PRD 로 추적되며, 픽셀·색·타이포는 규정하지 않는다(SPEC 소유). 로그인/회원/프로파일러/서버 측 사용자 쓰기 UI는 제품 범위에서 영구 제외이므로 어떤 와이어에도 없다. 회사 등록 대상은 복지 데이터가 있는 **~96개**이며, 기업유형 프리셋은 회사페이지 폴백이 아니라 비교 툴 **직접 입력 모드** 기본복지 템플릿용이다.

## 4. 광고 배치 요약 (page_type × 광고, FR-73)

| page_type | 대상 화면 | 자동광고 | 수동 슬롯 | 제휴 |
| --- | --- | :---: | --- | :---: |
| landing | SCR-10 | ON | 콘텐츠 하단 1 | 선택 |
| company | SCR-50 | ON | 본문 중간 1 + 하단 1 | ○ |
| combo | SCR-54 | ON | 본문 중간 1 + 하단 1 | ○ |
| **input** | SCR-20~34, 93 | **OFF** | **없음(무광고 강제, MON8)** | **없음** |
| result | SCR-40~43 | OFF | 리포트 하단 1개(절제, MON9) | 선택 |
| policy | SCR-60 | OFF/최소 | 없음 | 없음 |
| (셸·검색뷰·오류·404) | SCR-01·22·90·91·95 | OFF | 없음(안전 기본값) | 없음 |

- 모든 슬롯은 **예약 높이로 CLS ≤ 0.1**(FR-74), **"광고" 라벨** 상시 표기(FR-76·NFR19). 광고는 정적 본문 색인 이후 스크립트 주입(비-JS 본문 무손상).
- **비교 입력(input) 무광고 강제**·**결과(result) 하단 정확히 1개**가 UX 보호 핵심 규칙.

## 5. 커버리지 메모

- FLOW가 확정한 SCR 대역(SCR-10~13·20~24·30~34·40~43·50~56·60~62·90~95 + 공통 SCR-01·61·62)을 8개 문서가 1:1로 저충실도화했고, 신규 화면·구획을 창설하지 않았다.
- 공통 블록(헤더·푸터·동의 배너·광고 슬롯·제휴 카드·배지·버튼·폼)은 01이 정의하고 나머지 문서는 이름으로 인용한다.
- 회사·복지 데이터의 실제 목록·값은 빌드타임 큐레이션(복지 보유 ~96개)으로 SPEC/생성 스크립트가 확정하며, 와이어의 회사명·금액은 레이아웃 예시일 뿐이다.
