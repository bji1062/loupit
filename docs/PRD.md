# PRD — loupit

> 이 문서는 **얇은 인덱스**다. 요약과 참조 파일 목록, 요구사항 ID 마스터 인덱스만 담는다. 상세 서술은 각 `PRD/xx.md` 파일에 있다.

## 1. 제품 한 문단 요약

loupit(도메인 jobcho.wiki)은 **로그인 없는 한국 회사 복지·연봉·근무조건 비교 도구**이자 회사별 정보 페이지 모음이며, Google AdSense와 제휴(affiliate) 링크로 수익화한다. 기존 로그인 기반 풀스택 제품 `job_change`("직장 선택 OS")를 비밀번호·OAuth·소셜 로그인·소셜피드·프로파일러를 제거하고 광고 수익 모델로 재설계한 것으로, 코드는 새로 작성하되 큐레이션된 회사·복지 데이터(기업유형 6종·복지 데이터가 있는 회사 ~96개·상세 복지 SQL 96개 파일)는 보존한다. 사용자는 회원가입 없이 현직(a)과 이직처(b) 두 직장의 연봉·복지·근무형태·통근을 입력하고 우선순위(연봉/워라밸/복지)를 직접 선택하면, **100% 클라이언트 사이드**에서 총보상·시간가치·워라밸·복지 카테고리별 델타 리포트를 즉시 얻는다. 익명 사용자 데이터는 서버에 저장하지 않고 브라우저 localStorage에만 둔다. 백엔드는 익명 열람이 쓰기·인증 없는 **슬림 읽기 전용 FastAPI + MySQL**이며(복지 데이터 기여 쓰기 경로는 SC14로 분리), 회사 상세 ~96개(복지 데이터가 있는 회사)와 인기 비교 조합 페이지를 빌드타임에 정적 생성해 검색 색인·AdSense 승인 콘텐츠를 확보한다. 핵심 열람·비교는 로그인 없이 유지되고, 복지 데이터의 등록·수정(기여)에 한해 무비밀번호 이메일 코드 로그인과 회사 도메인 이메일 재직 인증을 요구한다(SC14, 2026-07-21 결정).

## 2. 목표 · 범위 핵심 요약

- **북극성 지표**: 검색 유입 기반 월 순방문(UV). 인과 사슬은 SEO 색인 확대 → organic UV 증가 → 광고 노출·수익 증가다.
- **In-scope(MVP)**: F1 회사 검색·선택, F2 비교 입력(우선순위 직접 선택), F3 클라이언트 비교 리포트, F4 회사 상세 정적 ~96개, F5 인기 비교 조합 정적 페이지, F6 광고·제휴 슬롯, F7 정책 페이지, F8 읽기 전용 API(익명 GET 5종 + 익명 비교 로그 POST 1종).
- **In-scope 증분(2026-07-21)**: SC14 인증 기여자의 복지 등록·수정 — 무비밀번호 이메일 코드 로그인 + 회사 도메인 이메일 재직 인증. 열람·비교는 로그인 불필요(익명 유지), 서버 쓰기는 기여 경로 한정.
- **Out-of-scope**: 프로파일러·비밀번호/OAuth/소셜 로그인·소셜피드는 **영구 제외**(SC9·SC10). 복지 가이드 콘텐츠·회사 랭킹/리스트·사용자 저장/신고는 2차 검토.
- **불변 제약**: Oracle Cloud ARM Always Free 단일 구성, 빌드 없는 바닐라 JS 프론트, **익명 경로는 읽기 전용 백엔드**(기여 쓰기는 SC14 한정·세션은 미들웨어 아닌 라우트 의존성), 계산 100% 클라이언트, **익명** 사용자 데이터 localStorage 한정.
- **릴리스 게이트**: 광고 "광고" 표기(공정위 규정), 정책 페이지 게시, AdSense 승인.
