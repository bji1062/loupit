격리 DB loupit_beta · 프로덕션 무관 · 이 페이지는 백엔드 API를 직접 호출합니다.

⚠️ 로그인 코드는 실제 메일 대신 서버 로그에 찍혀요. "코드 받기"를 누른 뒤 진행자(Claude)에게 코드를 물어보세요 — 로그에서 읽어 알려드립니다. (무비밀번호 이메일 코드 로그인)

① 로그인 (무비밀번호)

이메일 → 코드 받기 → 6자리 코드 → 로그인. 성공 시 세션 쿠키가 자동 설정돼요.

② 내 정보 / 로그아웃

로그인 후 세션 쿠키로 조회돼요. 닉네임·상태·재직 인증 목록.

③ 재직 인증 (도메인)

삼성전자=회사ID 40, 도메인 samsung.com. 회사 이메일로 코드 받고 인증.

④ 복지 편집 (재직 인증 필요)

③에서 회사 40 재직 인증 후 등록 가능. 배지는 서버가 verified로 강제해요.

⑤ 편집 이력 (공개·익명)

로그인 없이 누구나 조회. 닉네임만 노출(이메일·ID 미노출).
